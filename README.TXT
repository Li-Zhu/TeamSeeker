Distributed System for scheduling team games 
March 2015 
•	Implement Multi-threaded Clients-Server mode and complete user authentication module and game registration management.
•	Clients: check the gaming schedule, create gaming time and wait others to join, or join other players’ gaming time.
•	Server: send Clients different notification and gaming schedule base on receiving clients’ instructions.
•	This project requires good knowledge of socket and concurrent programming which is the core and hard part of basic Java.
